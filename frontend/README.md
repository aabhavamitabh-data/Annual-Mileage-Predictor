## 🎯 Overview
This is the frontend for the Annual Mileage Predictor, a ML-powered system that predicts how many kilometres a vehicle will be driven annually, used for insurance premium pricing.

## ⚡ Features
- Dark glassmorphism design
- Animated prediction results
- 4-tier risk classification display
- Income and vehicle age sliders
- Connects to FastAPI ML backend
- Fully responsive layout

## 🛠️ Tech Stack
| Technology | Purpose |
|---|---|
| Next.js 16 | React framework |
| Tailwind CSS | Styling |
| Framer Motion | Animations |
| shadcn/ui | UI components |
| lucide-react | Icons |

## 🚀 Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

Make sure the backend is running on http://localhost:8000
